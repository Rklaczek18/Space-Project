How fast is the space-discovery industry growing year over year? 
This project highlights the changes in the global space-discovery industry since 1957, specifically showing the growth over all the years. 
In the charts/graphs it can be seen that there was a peak in number of space missions, but then a decline, only peak again after that. 
It can be seen that the types of companies preforming these missions are scattered from companies all across the globe, and are not all specifically tied to one or any government.
This artifact analyzes the amount of space missions happening every year up until recent times. 
This artifact also analyzes the number of missions each individual company has contributed to the industry, highlighting the difference in credibility for each company. From that data, the audience can see that companies from all over the world, contribute to space exploration. Not only is the number of missions increasing, but the diversity of mission activity is high as well.
